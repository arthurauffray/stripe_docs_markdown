# Tokens

An issuing token object is created when an issued card is added to a digital wallet. As a [card issuer](https://docs.stripe.com/docs/issuing.md), you can [view and manage these tokens](https://docs.stripe.com/docs/issuing/controls/token-management.md) through Stripe.

## Endpoints

### Update a token status

- [POST /v1/issuing/tokens/:id](https://docs.stripe.com/api/issuing/tokens/update.md)

### Retrieve an issuing token

- [GET /v1/issuing/tokens/:id](https://docs.stripe.com/api/issuing/tokens/retrieve.md)

### List all issuing tokens for card

- [GET /v1/issuing/tokens](https://docs.stripe.com/api/issuing/tokens/list.md)
