# Physical Bundles

A Physical Bundle represents the bundle of physical items - card stock, carrier letter, and envelope - that is shipped to a cardholder when you create a physical card.

## Endpoints

### Retrieve a physical bundle

- [GET /v1/issuing/physical_bundles/:id](https://docs.stripe.com/api/issuing/physical_bundles/retrieve.md)

### List all physical bundles

- [GET /v1/issuing/physical_bundles](https://docs.stripe.com/api/issuing/physical_bundles/list.md)
