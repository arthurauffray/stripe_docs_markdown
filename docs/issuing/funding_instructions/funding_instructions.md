# Funding Instructions

Funding Instructions contain reusable bank account and routing information. Push funds to these addresses via bank transfer to [top up Issuing Balances](https://docs.stripe.com/docs/issuing/funding/balance.md).

## Endpoints

### Create funding instructions

- [POST /v1/issuing/funding_instructions](https://docs.stripe.com/api/issuing/funding_instructions/create.md)

### List all funding instructions

- [GET /v1/issuing/funding_instructions](https://docs.stripe.com/api/issuing/funding_instructions/list.md)

### Simulate a top up

- [POST /v1/test_helpers/issuing/fund_balance](https://docs.stripe.com/api/issuing/funding_instructions/fund.md)
