# Disputes

As a [card issuer](https://docs.stripe.com/docs/issuing.md), you can dispute transactions that the cardholder does not recognize, suspects to be fraudulent, or has other issues with.

Related guide: [Issuing disputes](https://docs.stripe.com/docs/issuing/purchases/disputes.md)

## Endpoints

### Create a dispute

- [POST /v1/issuing/disputes](https://docs.stripe.com/api/issuing/disputes/create.md)

### Update a dispute

- [POST /v1/issuing/disputes/:id](https://docs.stripe.com/api/issuing/disputes/update.md)

### Retrieve a dispute

- [GET /v1/issuing/disputes/:id](https://docs.stripe.com/api/issuing/disputes/retrieve.md)

### List all disputes

- [GET /v1/issuing/disputes](https://docs.stripe.com/api/issuing/disputes/list.md)

### Submit a dispute

- [POST /v1/issuing/disputes/:id/submit](https://docs.stripe.com/api/issuing/dispute/submit.md)
