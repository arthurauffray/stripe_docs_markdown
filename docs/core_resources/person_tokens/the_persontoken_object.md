# Person Tokens

Person Tokens are single-use tokens which tokenize person information, and are used for creating or updating a Person.

## Endpoints

### Create a person token

- [POST /v2/core/accounts/:id/person_tokens](https://docs.stripe.com/api/v2/core/accounts/create-person-token.md)

### Retrieve a person token

- [GET /v2/core/accounts/:id/person_tokens/:id](https://docs.stripe.com/api/v2/core/accounts/retrieve-person-token.md)
