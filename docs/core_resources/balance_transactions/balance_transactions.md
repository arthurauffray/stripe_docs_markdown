# Balance Transactions

Balance transactions represent funds moving through your Stripe account. Stripe creates them for every type of transaction that enters or leaves your Stripe account balance.

Related guide: [Balance transaction types](https://docs.stripe.com/docs/reports/balance-transaction-types.md)

## Endpoints

### Retrieve a balance transaction

- [GET /v1/balance_transactions/:id](https://docs.stripe.com/api/balance_transactions/retrieve.md)

### List all balance transactions

- [GET /v1/balance_transactions](https://docs.stripe.com/api/balance_transactions/list.md)
