# Account Tokens

Account tokens are single-use tokens which tokenize company/individual/business information, and are used for creating or updating an Account.

## Endpoints

### Create an account token

- [POST /v2/core/account_tokens](https://docs.stripe.com/api/v2/core/accounts/create-account-token.md)

### Retrieve an account token

- [GET /v2/core/account_tokens/:id](https://docs.stripe.com/api/v2/core/accounts/retrieve-account-token.md)
