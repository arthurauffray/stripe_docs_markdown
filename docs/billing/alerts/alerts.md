# Alerts

A billing alert is a resource that notifies you when a certain usage threshold on a meter is crossed. For example, you might create a billing alert to notify you when a certain user made 100 API requests.

## Endpoints

### Create a billing alert

- [POST /v1/billing/alerts](https://docs.stripe.com/api/billing/alert/create.md)

### Retrieve a billing alert

- [GET /v1/billing/alerts/:id](https://docs.stripe.com/api/billing/alert/retrieve.md)

### List billing alerts

- [GET /v1/billing/alerts](https://docs.stripe.com/api/billing/alert/list.md)

### Activate a billing alert

- [POST /v1/billing/alerts/:id/activate](https://docs.stripe.com/api/billing/alert/activate.md)

### Archive a billing alert

- [POST /v1/billing/alerts/:id/archive](https://docs.stripe.com/api/billing/alert/archive.md)

### Deactivate a billing alert

- [POST /v1/billing/alerts/:id/deactivate](https://docs.stripe.com/api/billing/alert/deactivate.md)
