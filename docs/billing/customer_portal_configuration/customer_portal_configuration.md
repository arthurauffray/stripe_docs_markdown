# Customer Portal Configuration

A portal configuration describes the functionality and behavior you embed in a portal session. Related guide: [Configure the customer portal](https://docs.stripe.com/customer-management/configure-portal.md).

## Endpoints

### Create a portal configuration

- [POST /v1/billing_portal/configurations](https://docs.stripe.com/api/customer_portal/configurations/create.md)

### Update a portal configuration

- [POST /v1/billing_portal/configurations/:id](https://docs.stripe.com/api/customer_portal/configurations/update.md)

### Retrieve a portal configuration

- [GET /v1/billing_portal/configurations/:id](https://docs.stripe.com/api/customer_portal/configurations/retrieve.md)

### List portal configurations

- [GET /v1/billing_portal/configurations](https://docs.stripe.com/api/customer_portal/configurations/list.md)
