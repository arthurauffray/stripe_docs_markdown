# Subscription Items

Subscription items allow you to create customer subscriptions with more than one plan, making it easy to represent complex billing relationships.

## Endpoints

### Create a subscription item

- [POST /v1/subscription_items](https://docs.stripe.com/api/subscription_items/create.md)

### Update a subscription item

- [POST /v1/subscription_items/:id](https://docs.stripe.com/api/subscription_items/update.md)

### Retrieve a subscription item

- [GET /v1/subscription_items/:id](https://docs.stripe.com/api/subscription_items/retrieve.md)

### List all subscription items

- [GET /v1/subscription_items](https://docs.stripe.com/api/subscription_items/list.md)

### Delete a subscription item

- [DELETE /v1/subscription_items/:id](https://docs.stripe.com/api/subscription_items/delete.md)
