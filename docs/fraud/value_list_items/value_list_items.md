# Value List Items

Value list items allow you to add specific values to a given Radar value list, which can then be used in rules.

Related guide: [Managing list items](https://docs.stripe.com/docs/radar/lists.md#managing-list-items)

## Endpoints

### Create a value list item

- [POST /v1/radar/value_list_items](https://docs.stripe.com/api/radar/value_list_items/create.md)

### Retrieve a value list item

- [GET /v1/radar/value_list_items/:id](https://docs.stripe.com/api/radar/value_list_items/retrieve.md)

### List all value list items

- [GET /v1/radar/value_list_items](https://docs.stripe.com/api/radar/value_list_items/list.md)

### Delete a value list item

- [DELETE /v1/radar/value_list_items/:id](https://docs.stripe.com/api/radar/value_list_items/delete.md)
