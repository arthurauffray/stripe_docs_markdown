# Value Lists

Value lists allow you to group values together which can then be referenced in rules.

Related guide: [Default Stripe lists](https://docs.stripe.com/docs/radar/lists.md#managing-list-items)

## Endpoints

### Create a value list

- [POST /v1/radar/value_lists](https://docs.stripe.com/api/radar/value_lists/create.md)

### Update a value list

- [POST /v1/radar/value_lists/:id](https://docs.stripe.com/api/radar/value_lists/update.md)

### Retrieve a value list

- [GET /v1/radar/value_lists/:id](https://docs.stripe.com/api/radar/value_lists/retrieve.md)

### List all value lists

- [GET /v1/radar/value_lists](https://docs.stripe.com/api/radar/value_lists/list.md)

### Delete a value list

- [DELETE /v1/radar/value_lists/:id](https://docs.stripe.com/api/radar/value_lists/delete.md)
