# Configuration

A Configurations object represents how features should be configured for terminal readers. For information about how to use it, see the [Terminal configurations documentation](https://docs.stripe.com/terminal/fleet/configurations-overview.md).

## Endpoints

### Create a Configuration

- [POST /v1/terminal/configurations](https://docs.stripe.com/api/terminal/configuration/create.md)

### Update a Configuration

- [POST /v1/terminal/configurations/:id](https://docs.stripe.com/api/terminal/configuration/update.md)

### Retrieve a Configuration

- [GET /v1/terminal/configurations/:id](https://docs.stripe.com/api/terminal/configuration/retrieve.md)

### List all Configurations

- [GET /v1/terminal/configurations](https://docs.stripe.com/api/terminal/configuration/list.md)

### Delete a Configuration

- [DELETE /v1/terminal/configurations/:id](https://docs.stripe.com/api/terminal/configuration/delete.md)
