# Account Links

Account Links are the means by which a Connect platform grants a connected account permission to access Stripe-hosted applications, such as Connect Onboarding.

Related guide: [Connect Onboarding](https://docs.stripe.com/docs/connect/custom/hosted-onboarding.md)

## Endpoints

### Create an account link

- [POST /v1/account_links](https://docs.stripe.com/api/account_links/create.md)
