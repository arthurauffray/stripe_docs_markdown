# Capabilities

This is an object representing a capability for a Stripe account.

Related guide: [Account capabilities](https://docs.stripe.com/docs/connect/account-capabilities.md)

## Endpoints

### Update an Account Capability

- [POST /v1/accounts/:id/capabilities/:id](https://docs.stripe.com/api/capabilities/update.md)

### Retrieve an Account Capability

- [GET /v1/accounts/:id/capabilities/:id](https://docs.stripe.com/api/capabilities/retrieve.md)

### List all account capabilities

- [GET /v1/accounts/:id/capabilities](https://docs.stripe.com/api/capabilities/list.md)
