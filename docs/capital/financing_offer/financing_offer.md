# Financing Offer

This is an object representing an offer of financing from Stripe Capital to a Connect subaccount.

## Endpoints

### Retrieve a given financing offer

- [GET /v1/capital/financing_offers/:id](https://docs.stripe.com/api/capital/financing_offers/retrieve.md)

### List financing offers

- [GET /v1/capital/financing_offers](https://docs.stripe.com/api/capital/financing_offers/list.md)

### Mark that a financing offer has been delivered

- [POST /v1/capital/financing_offers/:id/mark_delivered](https://docs.stripe.com/api/capital/financing_offers/mark_delivered.md)
