# Tax Code

[Tax codes](https://stripe.com/docs/tax/tax-categories) classify goods and services for tax purposes.

## Endpoints

### Retrieve a tax code

- [GET /v1/tax_codes/:id](https://docs.stripe.com/api/tax_codes/retrieve.md)

### List all tax codes

- [GET /v1/tax_codes](https://docs.stripe.com/api/tax_codes/list.md)
